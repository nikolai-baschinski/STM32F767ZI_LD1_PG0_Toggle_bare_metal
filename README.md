This us a bare metal project for the STM32F767ZI on a Nucleo-F767ZI.
It toggles the green LED (LD1) with 1 Hz and the output pin G0 with 50 Hz.

1. Clone the repo with git
git clone https://github.com/nikolai-baschinski/STM32F767ZI_LD1_PG0_Toggle_bare_metal.git
2. Install the STM32CubeIDE and import the project.
3. Connect the Nucleo-STM32F67ZI dev board to the USB interface of your PC.
4. Build and Run the project on the target.
